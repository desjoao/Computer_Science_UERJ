# Exercício de Casa 2 (EC2)

Faça a implementação dos algoritmos a seguir:

     - Warshall - Para cálculo do fechamento transitivo de um grafo representado por uma matriz de ajacências.
     - Prim - Para obtenção da árvore geradora mínima de um grafo ponderado conexo.
     - Floyd - Para determinação de caminhos mínimos entre todos os pares de vértices.
     - Bellman-Ford - Para cálculo de caminhos mínimos a partir de uma origem, com detecção de ciclos de peso negativo.

Os algoritmos podem ser implementados em C, C++ ou Python e você deve entregar um único arquivo fonte contendo todas as implementações (com a extensão da linguagem escolhida: .c, .cpp ou .py). Cada algoritmo deve ser implementado como uma função separada, reutilizando uma estrutura principal comum para leitura do grafo e exibição dos resultados.
